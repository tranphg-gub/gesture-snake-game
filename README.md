# 🐍 Gesture Snake Game

Trò chơi Snake cổ điển kết hợp với điều khiển bằng cử chỉ tay sử dụng IoT và webcam.

## Tính năng

- 🎮 Trò chơi Snake cơ bản với điều khiển bằng bàn phím
- 👆 Điều khiển bằng cử chỉ tay (Thumb Up/Down/Left/Right)
- 📊 Hệ thống điểm số và cấp độ
- 💾 Lưu high score
- 🎨 Giao diện game hiện đại với hiệu ứng

## Yêu cầu

- Node.js 14+
- Webcam (để sử dụng tính năng cử chỉ tay)
- Trình duyệt web hiện đại (Chrome, Firefox, Safari)
